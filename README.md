# RBAC-server

The graph edge-based RBAC server which only store relations instead of entities

## Feature

- [x] User Permission
- [x] RBAC
- [x] HRBAC
- [x] Access Inheritance
- [x] Fine grained(given permission without role)
- [x] Multiple role
- [x] List who has access to object

## Reserved words

namespace: role, user
relation: member, parent
